  ```
  # on edge device
  # environments: BOOTSTRAP=true(default)|fase, DAEMON_PORT(8088, http), ROUTER_PORT(5549, tcp)

  # on cloud
  # environments: MSG_PORT(5548, tcp), HTTP_PORT(8088, http)
  $ MSG_PORT=<EVCLOUDSVC_PORT> evcloudsvc
  ```
